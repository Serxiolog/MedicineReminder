# Мобильное приложение "Напоминалка о лекарствах"

## Описание
Мобильное приложение для Android, разработанное на .NET MAUI Blazor, предназначенное для управления приёмом лекарств с функцией напоминаний.

## Технологии
- .NET 8.0
- .NET MAUI Blazor
- C# 12
- Android API 21+

## Функциональность

### Основные возможности:
1. **Управление лекарствами**
   - Добавление новых лекарств с указанием названия, дозировки, частоты приёма
   - Редактирование существующих записей
   - Удаление лекарств
   - Просмотр списка всех лекарств

2. **Настройка напоминаний**
   - Установка времени приёма
   - Выбор дней недели для приёма
   - Указание периода приёма (дата начала и окончания)
   - Локальные push-уведомления

3. **Отслеживание приёма**
   - Просмотр предстоящих приёмов
   - Отметка о принятых лекарствах
   - Отображение пропущенных приёмов
   - Статистика по лекарствам

4. **Хранение данных**
   - Локальное хранение в JSON-файлах
   - Сохранение данных между запусками приложения
   - CRUD-операции для всех сущностей

## Установка и запуск

### Требования:
- Visual Studio 2022 (17.8+) с .NET MAUI workload
- Android SDK (API 21+)
- .NET 8.0 SDK

### Шаги:
1. Откройте проект в Visual Studio 2022
2. Восстановите NuGet пакеты
3. Выберите целевое устройство Android (эмулятор или физическое устройство)
4. Нажмите F5 для запуска

## Разрешения Android

Приложение запрашивает следующие разрешения:
- `POST_NOTIFICATIONS` - для отправки уведомлений (Android 13+)
- `SCHEDULE_EXACT_ALARM` - для точного планирования уведомлений
- `USE_EXACT_ALARM` - для использования точных будильников
- `RECEIVE_BOOT_COMPLETED` - для восстановления напоминаний после перезагрузки
- `WAKE_LOCK` - для пробуждения устройства при уведомлении

## Архитектура

### Модели данных:
- **Medicine**: хранит информацию о лекарстве (название, дозировка, время, частота, заметки)
- **Reminder**: хранит запланированные приёмы с привязкой к лекарству

### Сервисы:
- **DataService**: управление данными (CRUD операции, генерация напоминаний)
- **NotificationService**: работа с системными уведомлениями Android

### UI:
- Реализован на Blazor компонентах
- Адаптивный дизайн для мобильных устройств
- Интуитивный интерфейс с использованием emoji-иконок

## Особенности реализации

### Хранение данных:
- Используется локальное файловое хранилище через `FileSystem.AppDataDirectory`
- Данные сериализуются в JSON формат
- Автоматическое сохранение при изменениях

### Уведомления:
- Используется `AlarmManager` для планирования точных уведомлений
- `BroadcastReceiver` для обработки срабатывания будильника
- Поддержка Android 13+ с запросом разрешений

### Генерация напоминаний:
- Автоматическое создание напоминаний на основе расписания
- Учёт выбранных дней недели
- Поддержка периода приёма лекарств
